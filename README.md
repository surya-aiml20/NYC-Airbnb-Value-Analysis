# NYC Airbnb Neighbourhood Value Analysis

This Streamlit dashboard analyzes Airbnb value across New York City
using price, availability, and demand-based metrics.

## Features
- Value score computation
- Borough-based exploration
- Undervalued vs overpriced analysis
- Interactive Folium map
- Rule-based chatbot

## Tech Stack
- Python
- Pandas
- Streamlit
- Folium
- Seaborn
